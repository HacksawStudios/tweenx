TweenX

[日本語|English]

TweenX is a cross platform tween libraly for Haxe. It works on Haxe2 and Haxe3, which is targeting Flash, JavaScript, C++ and 32bit Neko.

Demo

[Flash]
[HTML5]

Feature

-
  • method chaining
  • 41 types of easing
  • delay
  • event handler
  • repeat, yoyo, zigzag
  • fast forward, backward, slow
  • function tween
  • tween with relative value
  • manual updating
  • serialization, parallelization
  • default value
  • RGB, HSV color tween
  • shaking
  • debug mode(-D tweenx_debug)

TweenX is for Flash like platform.

Speed

Frames per second, when 15000 objects moves on Flash

Tweenx is faster than Tweener, Actuate, Tween24 and Tweensy.

Default

						//default values of each parameters
						TweenX.defaultTime		= 0.3;
						TweenX.defaultEase		= EaseX.linear;
						TweenX.defaultDelay 	= 0;
						TweenX.defaultRepeat 	= 1;
						TweenX.defaultYoyo 		= false;
						TweenX.defaultZigZag 	= false;
						TweenX.defaultInterval 	= 0;
						TweenX.defaultAutoPlay 	= true;
						
						//get
						var defaults:DefaultsX = TweenX.dumpDefaults();
						
						//change
						defaults.ease( EaseX.bounceOut ).time( 1 );
						
						//apply
						TweenX.setDefaults( defaults );
						
						//initialize
						TweenX.initDefaults();
						
						//apply default values to serialization group
						TweenX.serial([
							TweenX.to( square ).x( 360 ),
							TweenX.to( square ).alpha( 0 ),
						], defaults );

Event Handler

						//on
						TweenX.to( square ).x( 360 )
							.onDelay( _delay ); 	
							.onHead( _head );		
							.onUpdate( _update );	
							.onFoot( _foot );		
							.onInterval( _interval );
							.onRepeat( _repeat );	
							.onRest( _rest );		
							.onFinish( _finish );	
							.onProgress( _progress );
							.onPlay( _play );		
							.onStop( _stop );		
							
						//addEventListener
							.addEventListener( EventX.DELAY, 	delay );
							.addEventListener( EventX.HEAD, 	head );
							.addEventListener( EventX.REPEAT, 	repeat );
							.addEventListener( EventX.FOOT, 	foot );
							.addEventListener( EventX.UPDATE, 	update );
							.addEventListener( EventX.INTERVAL, interval );
							.addEventListener( EventX.REST, 	rest );
							.addEventListener( EventX.FINISH, 	finish );
							.addEventListener( EventX.PLAY, 	play );
							.addEventListener( EventX.STOP, 	stop );

Play, Stop, Fast Forward, Backward, and other

						//Stop
						tween.stop();
						
						//Play
						tween.play();
						
						//Jump
						tween.goto( 0.1 );
						
						//Play from the top
						tween.goto( 0, true );
						
						//Current Time
						tween.currentTime;
						
						//Total Time
						tween.totalTime;
						
						//Fast Forward
						tween.timeScale = 8;
						
						//Backward
						tween.timeScale = -1;
						
						TweenX.topLevelTimeScale = 0.5;

Tween with Relative value

						TweenX.serial([
							//Below 3 tweens move in the same way
							
							//Absolute tween 
							TweenX.serial([
								TweenX.to( square ).xy( 360,   0 ),
								TweenX.to( square ).xy( 360, 360 ),
								TweenX.to( square ).xy(   0, 360 ),
								TweenX.to( square ).xy(   0,   0 ),
							]),
							TweenX.wait( 0.5 ),
							
							//Relative tween from current value.
							TweenX.serial([
								TweenX.to( square )._xy( 360,   0 ),
								TweenX.to( square )._xy( 360, 360 ),
								TweenX.to( square )._xy(   0, 360 ),
								TweenX.to( square )._xy(   0,   0 ),
							]),
							TweenX.wait( 0.5 ),
							
							//Relative tween from start value.
							TweenX.serial([
								TweenX.to( square ).__xy(  360,    0 ),
								TweenX.to( square ).__xy(    0,  360 ),
								TweenX.to( square ).__xy( -360,    0 ),
								TweenX.to( square ).__xy(    0, -360 ),
							]),
						]).delay( 0.5 );

Function Tween

TweenX has 5 function for Function Tween.

  • tweenFunc1( target, from1, to1 )
  • tweenFunc2( target, from1, to1, from2, to2 )
  • tweenFunc3( target, from1, to1, from2, to2, from3, to3 )
  • tweenFunc4( target, from1, to1, from2, to2, from3, to3, from4, to4 )
  • tweenFunc( target, fromArray, toArray )

    • 						function draw( x:Float, size:Float ) {
							graphics.lineStyle(1, 0x335F73 );
							graphics.drawCircle( x, 200, size );
						}
						
						TweenX.tweenFunc2( draw, 10, 10, 300, 100 ).time( 0.5 ).ease( EaseX.quadIn );
